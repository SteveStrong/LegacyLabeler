# Test Documents Folder

This folder contains sample documents for development and testing of the Legacy Labeler application.

## Supported File Types
- PDF files (.pdf)
- JPEG images (.jpg, .jpeg)
- PNG images (.png)
- TIFF images (.tiff)

## Usage
Place your test documents in this folder. The application will automatically scan this directory and make all supported files available for review and labeling.

## Notes
- Files can be organized in subfolders if needed
- The application performs read-only operations - your original files remain unchanged
- This is a development/testing setup - production versions will support external folders and cloud storage
